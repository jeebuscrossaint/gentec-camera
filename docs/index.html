Overview

The Gentec Beamage 4M is a professional IR beam profiling camera that ships with a C# SDK. This project wraps that SDK via pythonnet to expose a clean Python API, letting researchers capture 2048×2048 IR beam profiles, extract beam statistics, and export data without leaving their Python workflow.

Built for Dr. Eikenberry's group at UCF CREOL (Center for Research and Education in Optics and Lasers), one of the largest optics research centers in the US.

Features

  • Connect to and control Beamage-4M hardware over USB
  • Auto or manual exposure (0.06–5000 ms range)
  • 4-sigma beam diameter (X & Y), centroid position, and FPS — live from hardware
  • Export to FITS with full astronomical headers (timestamp, exposure, wavelength, beam stats)
  • PNG visualization with full-sensor view and auto-detected peak zoom
  • Thread-safe event-driven capture with configurable timeout
  • CLI with multi-frame capture, verbosity levels, and camera selection by serial number

Usage

# auto-exposure, capture 10 frames
python main.py --auto-exposure --frames 10

# manual exposure, custom output file
python main.py --exposure 50.5 --output capture.fits

# select camera by serial number
python main.py --serial 228451 --auto-exposure

# quiet mode, skip plot
python main.py --exposure 100 --frames 5 --no-plot

Stack

  • Python 3 — core implementation
  • pythonnet — bridges Python to the .NET BeamageSDK.dll
  • NumPy — image array processing
  • Matplotlib — beam visualization
  • Astropy — FITS file I/O
